Works the way you do

Stealth by default

The overlay is virtually invisible when idle. It never gets in your way — only appears when you need it.

Notion AI powered

Uses your existing Notion AI subscription. No separate API keys, no hidden costs — just your account.

Streaming & Markdown

Typewriter-style output with full markdown rendering — bold, italic, code blocks, and headings.

Get started in 5 minutes

1

Install notion2api

Clone the server, install dependencies, and configure your Notion token_v2.

2

Start the server

Run npm start in the notion2api directory. Keep it running in the background.

3

Launch HideAIChat

Open the app. Press ⌘⌥N to show the overlay and start chatting.
