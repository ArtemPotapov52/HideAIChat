What is dependency injection in Swift?
Dependency injection (DI) is a pattern where an object receives its dependencies from an external source rather than creating them internally.

In Swift, the most common form is initializer injection:
class Service {
  let api: APIClient
  init(api: APIClient) {
    self.api = api
  }
}

This makes code more testable and loosely coupled.

▸ Refactor Swift to async/await
"Rewrite this using async/await:"
func fetchUser(id: String, completion: @escaping (User?) -> Void) { URLSession.shared.dataTask(with: url) { data, _, _ in completion(data.flatMap { try? JSONDecoder().decode(User.self, from: $0) }) }.resume() }
→
func fetchUser(id: String) async throws -> User? { let (data, _) = try await URLSession.shared.data(from: url) return try? JSONDecoder().decode(User.self, from: data) }

Get started in 5 minutes

1

Install notion2api

Clone the server, install dependencies, and set your Notion token_v2 cookie in config.json.

2

Start the server

Run npm start in the notion2api directory. Keep it running on port 3123.

3

Launch HideAIChat

Open the app. Press ⌘⌥N to summon the overlay and chat with AI.
